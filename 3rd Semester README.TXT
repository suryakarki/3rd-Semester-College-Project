#Description of the program:

The program prints the numbers whose sum of individual digits and sum of square of individual digits are both prime numbers. The numbers exist between two numbers seperated by a ','; which are read from the input file. The first line of the file contains a number n which is the total number of lines after it.

Example Input:

2
1,20
120,130

Output:

11,12,14,16
120
